Job Description

We are looking for a Sr Full Stack Engineer to produce scalable software solutions. Will be part of a cross-functional team that’s responsible for the full software development life cycle, from conception to deployment.

The ideal candidate will have experience in cryptocurrency, as well as experience building browser extensions for chrome or firefox. You will be adding new features and optimizing existing systems on their wallet platform.

If you are familiar with Agile methodologies, we’d like to meet you.

Responsibilities

  • Build pixel-perfect, buttery smooth UI, UX across both mobile platforms and Web.
  • Build Scalable, Secure enterprise applications
  • Design and build mobile applications for both Android & iOS platform.
  • Leverage native APIs, SDKs and bridges for deep integrations with both mobile platforms.

Desired Skills and Experience

  • At least 5+ years in professional web application experience
  • Experience developing browser extensions for Chrome or Firefox
  • At least 1 year of crypto experience
  • Experience with React, React Native, and NodeJS
  • Ability to contribute to a remote team through clean git workflows, code reviews, and standups
